## ADDED Requirements

### Requirement: Chat Query Tool
The system SHALL provide an MCP tool `ask_notebook` to send a chat query to a specific notebook.
The tool SHALL accept optional parameters `source_ids` (to restrict the query to specific sources) and `conversation_id` (to continue an existing chat).

#### Scenario: User asks a question
- **WHEN** the `ask_notebook` tool is invoked with a `notebook_id` and `query`
- **THEN** the tool returns the text answer generated by NotebookLM along with citations and `conversation_id`

#### Scenario: User asks a question restricted by sources
- **WHEN** the `ask_notebook` tool is invoked with `notebook_id`, `query`, and a list of `source_ids`
- **THEN** the system limits its retrieval to only the specified sources when generating the answer

### Requirement: Chat History Tool
The system SHALL provide an MCP tool `get_chat_history` to retrieve the conversation history of a specific notebook.

#### Scenario: User gets chat history
- **WHEN** the `get_chat_history` tool is invoked with a `notebook_id`
- **THEN** the tool returns a list of past Q&A turns with the conversation ID

### Requirement: Save Chat Note Tool
The system SHALL provide an MCP tool `save_chat_note` to save text content as a note in the notebook.

#### Scenario: User saves a note
- **WHEN** the `save_chat_note` tool is invoked with a `notebook_id`, `title`, and `content`
- **THEN** the system creates a note inside the notebook with the content
