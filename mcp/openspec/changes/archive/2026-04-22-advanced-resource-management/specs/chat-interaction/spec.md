## MODIFIED Requirements

### Requirement: Chat Query Tool
The system SHALL provide an MCP tool `ask_notebook` to send a chat query to a specific notebook.
The tool SHALL accept optional parameters `source_ids` (to restrict the query to specific sources) and `conversation_id` (to continue an existing chat).

#### Scenario: User asks a question
- **WHEN** the `ask_notebook` tool is invoked with a `notebook_id` and `query`
- **THEN** the tool returns the text answer generated by NotebookLM along with citations

#### Scenario: User asks a question restricted by sources
- **WHEN** the `ask_notebook` tool is invoked with `notebook_id`, `query`, and a list of `source_ids`
- **THEN** the system limits its retrieval to only the specified sources when generating the answer
